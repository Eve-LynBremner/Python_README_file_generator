# GitHub README.md Generator using Python
        
## Description
This tool creates readme.md files for GitHub repos

## Installation
Clone the code locally. This should include: main.py, readme_content.py, readme_generator.py, readme_questions.py, requirements.txt. Make sure python is installed. Open your terminal (Git Bash). Create a virtual environment: `python -m venv myenv`. Activate the virtual environment: `source myenv/Scripts/activate`. Install dependencies using the requirements.txt file: `pip install -r requirements.txt`

## Usage
Run the main script in your terminal: `python main.py`. Follow the prompts in your console, answering all questions. A readme.md file will then be created in the current folder
